tup0 - simple text editor (inspired by ed).

REQUIREMENTS:
    [] be as SIMPLE as possible.
    [] commands similar to VIM.
    [] use buffer (arraylist).
    [] ":h"                     - help.
    [] ":i" or ":a"             - insert.
    [] ":a"                     - append.
    [] ":w [filename]"          - write.
    [] ":d[linenumber]"         - delete line.
    [] ":ex"                    - explore current directory (?)
    [] ":q"                     - quit.
